## Deployment Patterns

**AWS:**
- Serverless: Lambda + API Gateway + DynamoDB
- Containers: ECS Fargate + ALB
- Static Sites: S3 + CloudFront + Route53
- Full-Stack: Amplify or Elastic Beanstalk

**Azure:**
- Serverless: Azure Functions + API Management
- Containers: Container Apps or AKS
- Static Sites: Static Web Apps
- Full-Stack: App Service

**Multi-Cloud:**
- Terraform for unified IaC
- Consistent CI/CD patterns
- Cloud-agnostic application design
