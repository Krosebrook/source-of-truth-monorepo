- CQRS (Command Query Responsibility Segregation)

**Communication:**
- Synchronous: REST, gRPC
- Asynchronous: Message queues, Event streams

**Data Management:**
- Database per service
- Shared database (when appropriate)
- Event-driven data sync
- Cache aside pattern
